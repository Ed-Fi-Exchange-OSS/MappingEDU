# User Guide

This documentation describes concepts and techniques necessary for a
technical professional to understand and leverage MappingEDU, a system
for creating business logic mappings between two data standards.

## Audience

This documentation is for technical professionals who work
with educational database systems, including analysts, database
administrators and software developers.
