Entity to Entity Hinting

Use Entity Hinting to create 1-to-1 matches between analogous entities in the source and target Data Standards. For example, when the 'Staff' entity in one Standard is equivalent to the 'K12 Staff' entity in the other Standard, you can create that association here and MappingEDU will provide better matches and suggestions. When Entity Hints are provided:

  • The Matchmaker Browse and Search will pre-filter to the target entity.
  • The Matchmaker Suggest feature will rank target entity matches higher.
