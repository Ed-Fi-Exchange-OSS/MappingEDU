Entity Hinting

Use Entity Hinting to create 1-to-1 matches between analogous entities in the source and target Data Standards. For example, when the 'Staff' entity in one Standard is equivalent to the 'K12 Staff' entity in the other Standard, you can create that association here and MappingEDU will provide better matches and suggestions. Project Synonyms

Use Synonyms to create a thesaurus for your Mapping Project. For example, if 'Begin Date' in one Standard means the same thing as 'Start Date' in another Standard, or if 'Student' in one Standard is the same concept as 'K12 Student' in another Standard, you can create those Synonym entries here. Adding Synonyms helps to improve suggestions offered by the Suggest tab in the Matchmaker.

Templates

Use templates for when business logic is being repeated on multiple elements. Once added there a side panel on the business logic with a list of templates or access them by typing '#'.
