Element Groups

Element groups are an optional, top-level organization for Data Standards entered into MappingEDU. These are typically not part of the formal specification, but rather a logical grouping useful for data mapping purposes. For example, a state with many separate data collections might organize each collection file to be mapped as an element group, a vendor might organize API endpoints into Element Groups, and so forth.
